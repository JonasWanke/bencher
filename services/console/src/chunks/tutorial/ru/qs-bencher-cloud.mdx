## Создайте аккаунт в Bencher Cloud

Bencher - это открытое и само-хостимое программное обеспечение.
Если вас интересует само-хостинг, ознакомьтесь с [учебником по Docker для Bencher](/ru/docs/tutorial/docker/).
Но для этого учебника мы будем использовать Bencher Cloud.


### <a id="signup-for-bencher-cloud" href="/auth/signup" target="_blank">Зарегистрируйтесь в Bencher Cloud</a>
<br />

После создания аккаунта, вам нужно будет подтвердить свой адрес электронной почты.
Проверьте свою почту на наличие ссылки для подтверждения.
После этого вы должны быть вошли в систему в Bencher Cloud.
