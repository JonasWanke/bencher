Bencher is open source and self-hostable.
If you are interested in using Bencher Cloud, check out the [Bencher Cloud Quick Start tutorial](/docs/tutorial/quick-start/).
This tutorial will get you setup using Bencher Self-Hosted with Docker.
Then you can continue on to the [Quick Start tutorial](/docs/tutorial/quick-start/)
using your local instance of Bencher instead of Bencher Cloud.
