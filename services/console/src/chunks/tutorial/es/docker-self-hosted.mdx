Bencher es de código abierto y autoalojable.
Si estás interesado en usar Bencher Cloud, consulta el [tutorial rápido de inicio de Bencher Cloud](/docs/es/tutorial/quick-start/).
Este tutorial te permitirá configurar Bencher Self-Hosted con Docker.
Luego puedes continuar con el [tutorial rápido de inicio](/docs/es/tutorial/quick-start/)
usando tu instancia local de Bencher en lugar de Bencher Cloud.
