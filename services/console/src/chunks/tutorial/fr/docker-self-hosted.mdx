Bencher est open source et peut être hébergé par soi-même.
Si vous êtes intéressé par l'utilisation de Bencher Cloud, consultez le [tutoriel de démarrage rapide de Bencher Cloud](/docs/fr/tutorial/quick-start/).
Ce tutoriel vous aidera à vous installer en utilisant Bencher auto-hébergé avec Docker.
Ensuite, vous pouvez continuer avec le [tutoriel de démarrage rapide](/docs/fr/tutorial/quick-start/)
en utilisant votre instance locale de Bencher au lieu de Bencher Cloud.
