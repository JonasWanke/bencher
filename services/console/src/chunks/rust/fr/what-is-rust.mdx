## Qu'est-ce que Rust?

Rust est un langage de programmation open source qui met l'accent sur la vitesse, la fiabilité et la productivité. Il parvient à garantir la sécurité de la mémoire sans avoir besoin d'un ramasse-miettes.

Vous devriez envisager d'utiliser Rust si vous écrivez un :
- Programme de bas niveau où la performance est importante
- Bibliothèque partagée qui sera utilisée par plusieurs langages différents
- Interface de ligne de commande complexe (CLI)
- Projet logiciel à long terme avec de nombreux contributeurs

Rust met fortement l'accent sur la productivité des développeurs. Cargo est le gestionnaire de paquets officiel, et il gère de nombreuses tâches telles que :
- Gestion des dépendances du projet
- Compilation de binaires, tests et benchmarks
- Linting
- Formatage
