### Windows

<CliWindowsScript />

<br />

> 🐰 만약 `running scripts is disabled on this system`이라는 오류 메세지가 나타나면:
> - `관리자 권한으로 Powershell 열기`
> - Run: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned`
> - Enter: `Y`
> - 이 스크립트를 다시 실행하세요
