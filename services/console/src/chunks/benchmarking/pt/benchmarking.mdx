## O que é Benchmarking?

Benchmarking é a prática de testar o desempenho do seu código
para ver quão rápido (latência) ou quanta (vazão) trabalho ele pode realizar.
Esta etapa frequentemente negligenciada no desenvolvimento de software é crucial
para criar e manter códigos rápidos e de alto desempenho.
O Benchmarking fornece as métricas necessárias para os desenvolvedores entenderem
quão bem o seu código se comporta sob diversas cargas de trabalho e condições.
Pelos mesmos motivos pelos quais você escreve testes unitários e de integração para prevenir regressões de recursos,
você deve escrever benchmarks para prevenir regressões de desempenho.
Bugs de desempenho são bugs!
