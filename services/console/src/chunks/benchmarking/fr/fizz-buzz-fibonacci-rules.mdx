Les règles du FizzBuzzFibonacci sont les suivantes :

> Écrivez un programme qui imprime les entiers de `1` à `100` (inclus) :
> - Pour les multiples de trois, imprimez `Fizz`
> - Pour les multiples de cinq, imprimez `Buzz`
> - Pour les multiples de trois et cinq, imprimez `FizzBuzz`
> - Pour les nombres qui font partie de la séquence de Fibonacci, imprimez uniquement `Fibonacci`
> - Pour tous les autres, imprimez le nombre

La [séquence de Fibonacci](https://fr.wikipedia.org/wiki/Suite_de_Fibonacci) est une séquence dans laquelle chaque nombre est la somme des deux précédents.
Par exemple, en commençant par `0` et `1`, le prochain nombre dans la séquence de Fibonacci serait `1`.
Suivi par : `2`, `3`, `5`, `8` et ainsi de suite.
Les nombres qui font partie de la séquence de Fibonacci sont connus sous le nom de nombres de Fibonacci. Nous allons donc devoir écrire une fonction qui détecte les nombres de Fibonacci.
