## Qu'est-ce que le Benchmarking?

Le benchmarking est la pratique consistant à tester la performance de votre code pour voir à quelle vitesse (latence) ou combien (débit) de travail il peut accomplir. Cette étape souvent négligée dans le développement de logiciels est essentielle pour créer et maintenir un code rapide et performant. Le benchmarking fournit les mesures nécessaires pour que les développeurs comprennent comment leur code se comporte sous diverses charges de travail et conditions. Pour les mêmes raisons que vous écrivez des tests unitaires et d'intégration pour prévenir les régressions de fonctionnalités, vous devriez écrire des benchmarks pour prévenir les régressions de performance. Les bugs de performance sont des bugs!
