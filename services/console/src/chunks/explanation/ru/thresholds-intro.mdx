Пороговые значения позволяют вам отлавливать регрессии производительности в Bencher.
Пороговое значение присваивается уникальной комбинации: Ветка, Тестовая площадка и Мера.
Пороговое значение должно иметь нижнюю границу, верхнюю границу или обе.
Каждая граница используется для расчета лимита границы.
Затем каждый новый показатель проверяется по каждому лимиту границы.

- Нижняя граница
  - Нижняя граница используется, когда меньшее значение указывает на регрессию производительности, как в случае с Measure Throughput.
- Верхняя граница
  - Верхняя граница используется, когда большее значение указывает на регрессию производительности, как в случае с Measure Latency.
