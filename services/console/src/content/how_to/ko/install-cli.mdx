## GitHub Actions

[GitHub Action](https://github.com/marketplace/actions/bencher-cli)을 사용하여 Bencher CLI를 설치하세요.
더 자세한 내용은 [GitHub Actions 사용 방법](/docs/how-to/github-actions/)을 참조하세요.

<CliGitHubActions />

선택적으로 `version` 인수를 사용하여 사용하려는 Bencher CLI의 정확한 버전을 지정할 수 있습니다.
그렇지 않으면 최신 CLI 버전이 기본으로 사용됩니다.

<CliGitHubActionsVersion />

Bencher *Self-Hosted* 를 사용하는 경우 정확한 버전을 명시하세요.
아직까지 이벤트 변환 변화가 있기 때문에 Bencher *Cloud* 를 사용하는 경우에는 정확한 버전을 명시하지 __마세요__.
