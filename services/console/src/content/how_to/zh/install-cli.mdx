## GitHub Actions

使用[GitHub Action](https://github.com/marketplace/actions/bencher-cli) 安装Bencher CLI。
查看[如何使用GitHub Actions](/zh/docs/how-to/github-actions/)了解更多详情。

<CliGitHubActions />

有一个可选的`version`参数来指定一个确切的Bencher CLI的版本来使用。
否则，它将默认使用最新的CLI版本。

<CliGitHubActionsVersion />

如果使用Bencher *Self-Hosted*，请指定一个确切的版本。
如果使用Bencher *Cloud*，请__不要__指定一个确切的版本，因为仍然会有偶尔的破坏性的更改。
