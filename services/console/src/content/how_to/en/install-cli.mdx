## GitHub Actions

Install the Bencher CLI using the [GitHub Action](https://github.com/marketplace/actions/bencher-cli).
See [how to use GitHub Actions](/docs/how-to/github-actions/) for more details.

<CliGitHubActions />

There is an optional `version` argument to specify an exact version of the Bencher CLI to use.
Otherwise, it will default to using the latest CLI version.

<CliGitHubActionsVersion />

Specify an exact version if using Bencher *Self-Hosted*.
Do __not__ specify an exact version if using Bencher *Cloud* as there are still occasional breaking changes.
